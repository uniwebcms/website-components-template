# Block

A component designed to showcase multiple features.

<Gallery images={['block1.png']} />{' '}

## Expected content

-   Image: lead image as background (optional)
-   H2: lead heading (optional)
-   H1: main heading (recommended)
-   Paragraph: a single paragraph as body text (optional)
-   Image or video: trailing image/video as foreground asset (optional)

## Layout Options

-   Align the main heading (H1) to Left / Center / Right to activate different layouts
